# Title and Purpose
Title of Project: QAM2 Java Application
Title of Application: Appointment Scheduling Application
Purpose: The main purpose for this application is to allow the user to schedule appointments for customers.
It allows the user to add, modify, delete customer profiles. It also allows the user to add, modify, and delete appointments for given customers.

#IDE and Drivers

IntelliJ 2021.1.1
JDK 17.0.1
JavaFX javafx-sdk-17.0.1
MySQL driver: mysql-connector-java-8.0.25

#HOW TO

Load application. Provide login credentials upon LoginScreen execution. Once inside the application, use the buttons (Appointment, Customer, Reports) to go to the area of the application you would like to go.
Appointments leads to the area to add, modify, or delete appointments.
Customer leads to the area to add, modify, or delete customers.
(Customer cannot be deleted if appointments are assigned to them.)
Reports leads to the area to view the reports available. Brief description of reports are posted on the related buttons.
Current reports include: total appointments by month and contact, scheduled appointments by contact, and scheduled appointments for given customer.
